iclbench.prompt_builder.create_prompt_builder(config)[source]ÔÉÅ

Creates an instance of a prompt builder based on the provided configuration.

This function initializes a prompt builder by extracting relevant configuration parameters. It can be extended or modified to support different types of prompt builders beyond just the HistoryPromptBuilder.

Parameters:

config (Config) ‚Äì An object containing configuration settings, which must include the following keys: - max_history (int): Maximum number of text history entries to retain. - max_image_history (int): Maximum number of image history entries to retain. - max_cot_history (int): Maximum number of chain-of-thought history entries to retain.

Returns:

An instance of a prompt builder configured with the specified

history limits and any additional parameters defined in the config.

Return type:

PromptBuilder
