class iclbench.evaluator.Evaluator(env_name, config, original_cwd='')[source]ÔÉÅ

Bases: object

Class to evaluate an agent on a set of tasks in a given environment.

The Evaluator class is responsible for orchestrating the evaluation of agents across multiple tasks within a specified environment. It manages the setup of the environment, runs episodes, logs results, and can execute evaluations in parallel or sequentially.

Variables:

  • env_name (str) ‚Äì Name of the environment in which the agent operates.

  • config (Config) ‚Äì Configuration object containing evaluation parameters.

  • tasks (list) ‚Äì List of tasks for the specified environment.

  • num_episodes (int) ‚Äì Number of episodes to run for each task.

  • num_workers (int) ‚Äì Number of parallel worker processes to use.

  • max_steps_per_episode (int) ‚Äì Maximum number of steps per episode.

  • dataset (InContextDataset) ‚Äì Dataset object for managing in-context learning tasks.
